If n has k trailing zeros, then n² has 2k trailing zeros — always exactly double.
Consequence: A perfect square can only have 0, 2, 4, 6… (an even number of) trailing zeros. Any number ending in exactly 1, 3, or 5 zeros is definitely NOT a perfect square.

The gap between consecutive squares is always an odd number. (n+1)² − n² = 2n+1 This is the (n+1)th odd number — which is exactly why adding consecutive odd numbers builds squares!
