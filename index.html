The key insight
Each square n² is built from n L-shaped rings called gnomons. Ring k has 2k−1 cells. The sum 1+3+5+…+(2n−1) = n². Click any ring to isolate it.

The algebraic reason
(n+1)² = n² + (2n+1). Each new square adds the next odd number. So building squares by adding odd rings is the geometric proof of this identity.

Only 6 endings possible
Digits 2, 3, 7, 8 never appear as the last digit of any perfect square. Click a digit to see its mapping. Pairs like 1 & 9 always give the same output.

The shortcut
If a number ends in 2, 3, 7 or 8 — it cannot be a perfect square. No calculation needed. This is the fastest way to rule out non-squares.

The rule
n = a×10ᵏ → n² = a²×10²ᵏ. The exponent k doubles. So 1 zero → 2 zeros, 3 zeros → 6 zeros. Odd zero counts are impossible in perfect squares.
