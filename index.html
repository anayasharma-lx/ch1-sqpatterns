Square Number Patterns

Three patterns that let you instantly spot — or rule out — perfect squares.

Build a square by stacking L-shaped rings. Each ring — called a gnomon — is an odd number of cells. The first is 1, then 3, then 5… The sum of the first n odd numbers is always exactly n². Click any ring in the legend to highlight it.

Why? Going from n² to (n+1)² adds 2n+1 cells — always an odd number. This is the (n+1)-th odd number. So building squares by adding odd rings is not a coincidence — it's the algebraic identity (n+1)² = n² + (2n+1) made geometric.
Only 6 endings are possible for perfect squares. Click any input digit (0–9) to see which ending its square produces. Notice that 2, 3, 7, 8 never appear as outputs — and that pairs like 1 & 9 always produce the same ending.

The shortcut: If a number ends in 2, 3, 7, or 8 — it cannot be a perfect square. You don't need to calculate anything. This instantly rules out roughly 40% of all numbers.
Trailing zeros always double when you square. If n ends in k zeros, then n² ends in 2k zeros — because squaring multiplies the zero-tail by 2. This means a perfect square can only have an even number of trailing zeros.
